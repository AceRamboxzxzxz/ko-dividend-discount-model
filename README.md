# Coca-Cola (KO) Dividend Discount Model

## Overview
This project values Coca-Cola (KO) using the Gordon Growth Dividend Discount Model (DDM) built entirely in Excel.

The model estimates the intrinsic value of KO stock based on expected future dividends, assuming constant long-run dividend growth.

## Methodology
- Historical quarterly dividends aggregated into annual dividends
- Dividend growth estimated using historical averages
- Required return estimated using CAPM
- Intrinsic value calculated using the Gordon Growth Model

- Under conservative dividend-based assumptions, KO appears overvalued relative to intrinsic value

## Limitations
- Assumes constant perpetual dividend growth
- Highly sensitive to growth and discount rate assumptions
- Best suited for mature, dividend-paying companies
